                                  # B

                                  # 以 B 为首的属性

                                  #::backdropℹ️

                                  伪元素，在任何处于全屏模式的元素下的即刻渲染的盒子

                                  #backdrop-filterℹ️

                                  为一个元素后面区域添加图形效果（如模糊或颜色偏移）。因为它适用于元素背后的所有元素，为了看到效果，必须使元素或其背景至少部分透明

                                  #backface-visibilityℹ️

                                  指定当元素背面朝向观察者时是否可见

                                  #backgroundℹ️

                                  CSS 简写属性，用于一次性集中定义各种背景属性

                                  #background-attachmentℹ️

                                  决定背景图像的位置是在视口内固定，或者随着包含它的区块滚动

                                  #background-blend-modeℹ️

                                  定义该元素的背景图片，以及背景色如何混合

                                  #background-clipℹ️

                                  设置元素的背景（背景图片或颜色）是否延伸到边框、内边距盒子、内容盒子下面

                                  #background-colorℹ️

                                  设置元素的背景色，属性的值为颜色值或关键字 transparent 二者选其一

                                  #background-imageℹ️

                                  为一个元素设置一个或者多个背景图像

                                  #background-originℹ️

                                  规定了指定背景图片 background-image 属性的原点位置的背景相对区域

                                  #background-positionℹ️

                                  为每一个背景图片设置初始位置。这个位置是相对于由 background-origin 定义的位置图层的

                                  #background-position-xℹ️

                                  设置水平方向的位置，与每个背景图片等位置层设置属性 background-origin 相关

                                  #background-position-yℹ️

                                  设置初始状态时背景图片在垂直方向上的位置，这个位置相对于通过 background-origin 定义的背景层的原点进行定位

                                  #background-repeatℹ️

                                  定义背景图像的重复方式。背景图像可以沿着水平轴，垂直轴，两个轴重复，或者根本不重复

                                  #background-sizeℹ️

                                  设置背景图片大小

                                  #<basic-shape>ℹ️

                                  表现基础图形的 CSS 数据类型，作用于 clip-path 与 shape-outside 属性中

                                  #::before(:before)ℹ️

                                  伪元素，匹配选中的元素的第一个子元素。常通过 content 属性来为一个元素添加修饰性的内容。此元素默认为行内元素

                                  #:blankℹ️

                                  伪类，选择用户输入为空的输入框

                                  #<blend-mode>ℹ️

                                  CSS 数据类型，用于描述当元素重叠时，颜色应当如何呈现。它被用于 background-blend-mode 和 mix-blend-mode 属性

                                  #block-sizeℹ️

                                  根据元素的书写模式定义了元素块的横向或纵向尺寸

                                  #blur()ℹ️

                                  CSS 方法，将高斯模糊应用于输出图片

                                  #borderℹ️

                                  设置各种单独的边界属性的简写属性

                                  #border-blockℹ️

                                  CSS 简写属性，用于在样式表中的某处同时设置逻辑块向边框的各属性值

                                  #border-block-colorℹ️

                                  定义了元素的逻辑块向的边框颜色

                                  #border-block-endℹ️

                                  CSS 简写属性，在样式表中的某处同时设置逻辑块末边框的各属性值

                                  #border-block-end-colorℹ️

                                  定义了元素的逻辑块末的边框颜色，并根据元素的书写模式、行内方向和文本朝向对应至实体边框颜色

                                  #border-block-end-styleℹ️

                                  定义了元素的逻辑块末的边框线型，并根据元素的书写模式、行内方向和文本朝向对应至实体边框线型

                                  #border-block-end-widthℹ️

                                  定义了元素的逻辑块末的边框宽度，并根据元素的书写模式、行内方向和文本朝向对应至实体边框宽度

                                  #border-block-startℹ️

                                  CSS 简写属性，用于在样式表中的某处同时设置逻辑块首边框的各属性值

                                  #border-block-start-colorℹ️

                                  定义了元素的逻辑块首的边框颜色，并根据元素的书写模式、行内方向和文本朝向对应至实体边框颜色

                                  #border-block-start-styleℹ️

                                  定义了元素的逻辑块首的边框线型，并根据元素的书写模式、行内方向和文本朝向对应至实体边框线型

                                  #border-block-start-widthℹ️

                                  定义了元素的逻辑块首的边框宽度，并根据元素的书写模式、行内方向和文本朝向对应至实体边框宽度

                                  #border-block-styleℹ️

                                  定义了元素的逻辑块向的边框线型，并根据元素的书写模式、行内方向和文本朝向对应至实体边框线型

                                  #border-block-widthℹ️

                                  定义了元素的逻辑块向的边框宽度，并根据元素的书写模式、行内方向和文本朝向对应至实体边框宽度

                                  #border-bottomℹ️

                                  CSS 简写属性，设置元素下边框

                                  #border-bottom-colorℹ️

                                  设置一个元素底部边框的颜色

                                  #border-bottom-left-radiusℹ️

                                  设置元素左下角的圆角

                                  #border-bottom-right-radiusℹ️

                                  设置元素右下角的圆角

                                  #border-bottom-styleℹ️

                                  设置元素底部边框的线条样式

                                  #border-bottom-widthℹ️

                                  设置元素的底部边框宽度

                                  #border-collapseℹ️

                                  设置表格的边框是分开的还是合并的

                                  #border-colorℹ️

                                  CSS 简写属性，设置元素四个边框颜色

                                  #border-end-end-radiusℹ️

                                  定义了元素的逻辑边框半径，并根据元素的书写模式、行内方向和文本朝向对应至实体边框半径

                                  #border-end-start-radiusℹ️

                                  定义了元素的逻辑边框半径，并根据元素的书写模式、行内方向和文本朝向对应至实体边框半径

                                  #border-imageℹ️

                                  为元素的边框设置图像样式

                                  #border-image-outsetℹ️

                                  定义边框图像可超出边框盒的大小

                                  #border-image-repeatℹ️

                                  定义图片如何填充边框

                                  #border-image-sliceℹ️

                                  将使用 border-image-source 引用的图像划分为多个区域

                                  #border-image-sourceℹ️

                                  声明元素的边框图片 border-image 的资源

                                  #border-image-widthℹ️

                                  指定了 `border image` 的宽度

                                  #border-inlineℹ️

                                  CSS 简写属性，用于在样式表中的某处同时设置逻辑行向边框的各属性值

                                  #border-inline-colorℹ️

                                  定义了元素的逻辑行向的边框颜色，并根据元素的书写模式、行内方向和文本朝向对应至实体边框颜色

                                  #border-inline-endℹ️

                                  CSS 简写属性，用于在样式表中的某处同时设置逻辑行末边框的各属性值

                                  #border-inline-end-colorℹ️

                                  定义了元素的逻辑行向的边框宽度，并根据元素的书写模式、行内方向和文本朝向对应至实体边框宽度

                                  #border-leftℹ️

                                  CSS 简写属性，设置元素左边框样式

                                  #border-left-colorℹ️

                                  设置元素的左边框颜色

                                  #border-left-styleℹ️

                                  设置元素左边框的线条样式

                                  #border-left-widthℹ️

                                  设置元素左边框的宽度

                                  #border-radiusℹ️

                                  设置元素的外边框圆角

                                  #border-rightℹ️

                                  CSS 简写属性，设置元素右边框样式

                                  #border-right-colorℹ️

                                  设置元素的右边框颜色

                                  #border-right-styleℹ️

                                  设置元素右边框的线条样式

                                  #border-right-widthℹ️

                                  设置元素右边框的宽度

                                  #border-spacingℹ️

                                  指定相邻单元格边框之间的距离（只适用于 边框分离模式 border-collapse ）

                                  #border-start-end-radiusℹ️

                                  定义了元素的逻辑边框半径，并根据元素的书写模式、行内方向和文本朝向对应至实体边框半径

                                  定义了元素的逻辑边框半径，并根据元素的书写模式、行内方向和文本朝向对应至实体边框半径

                                  #border-styleℹ️

                                  CSS 简写属性，用来设定元素所有边框的样式

                                  #border-topℹ️

                                  CSS 简写属性，设置元素上边框样式

                                  #border-top-colorℹ️

                                  设置元素的上边框颜色

                                  #border-top-styleℹ️

                                  设置元素右边框的线条样式

                                  #border-top-widthℹ️

                                  设置元素上边框的宽度

                                  #border-top-left-radiusℹ️

                                  设置元素左上角的圆角效果

                                  #border-top-right-radiusℹ️

                                  属性设置元素的右上角的圆角效果

                                  #border-widthℹ️

                                  设置盒子模型的边框宽度

                                  #bottomℹ️

                                  定义了定位元素下外边距边界与其包含块下边界之间的偏移，非定位元素设置此属性无效（ bottom 的效果取决于元素的 position 属性）

                                  #box-decoration-breakℹ️

                                  定义当元素跨多行、多列或多页时，元素的片段应如何呈现

                                  #box-shadowℹ️

                                  在元素的框架上添加阴影效果

                                  #box-sizingℹ️

                                  定义了 user agent 应该如何计算一个元素的总宽度和总高度

                                  #break-afterℹ️

                                  设置在生成的盒子之后的页面，列或区域中断行为（换句话说，如何以及是否中断）。如果没有生成的盒子，则该属性将被忽略

                                  #break-beforeℹ️

                                  定义页面，列或区域在生成的盒子之前应如何处理中断。如果没有生成的盒子，则忽略该属性

                                  #break-insideℹ️

                                  设置在多列布局页面下的内容盒子如何中断，如果多列布局没有内容盒子，这个属性会被忽略

                                  #brightness()ℹ️

                                  CSS 函数，将线性乘数应用于输入图像，使其看起来更亮或更暗
